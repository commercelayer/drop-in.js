# `cl-add-to-cart`

Add a buy button to any page with embedded validation rules.

<Canvas withSource="open">
  <Story story={Basic} name="Basic" />
</Canvas>

<ArgsTable story="Basic" />

### Disabled

To enable the **add to cart** button, you'll need to have a `sku` attribute and a `quantity` value greater than 0.

The button below is disabled since there's no `sku` nor `quantity`.

<Canvas withSource="open">
  <Story story={WithoutAttributes} name="Without Attributes" />
</Canvas>

### Out Of Stock

`cl-add-to-cart` checks the inventory for the provided sku. When the product is out of stock the button will be disable.

<Canvas withSource="open">
  <Story story={OutOfStock} name="Out Of Stock" />
</Canvas>

### Nonexisting

The provided `sku` must exists. If nonexisting the add to cart button will stay disabled.

<Canvas withSource="open">
  <Story story={Nonexisting} name="Nonexisting" />
</Canvas>

### No Overselling

The product with sku *BACKPACKFFFFFF000000XXXX* has less than `100` stock unit, so the button with be disabled.
