# `cl-availability`

Display an availability message with localized delivery lead time and cost.

The `cl-availability` component is the owner of the availability information.
It has the responsibility to fetch the inventory of the provided sku and dispatch this information to its [**cl-availability-status**](?path=/docs/components-availability-cl-availability-status--basic) and [**cl-availability-message**](?path=/docs/components-availability-cl-availability-message--basic) children.

### No SKU? Empty box

`sku` attribute is mandatory. If you don't set it, the component will be rendered as an empty box without any size.
