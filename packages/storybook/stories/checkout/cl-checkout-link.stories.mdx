# `cl-checkout-link`

Any Commerce Layer account comes with a [hosted version of the Checkout](https://github.com/commercelayer/commercelayer-react-checkout#hosted-version) application that is automatically enabled.

<p>
  You can use the hosted version of the Checkout application with the following URL format:
  <span class="px-2 py-1 border border-[#EEEEEE] bg-[#F8F8F8] rounded text-[#464646] !text-[13px] !font-mono whitespace-nowrap inline-block">
    https://<b>&lt;your-organization-slug&gt;</b>.checkout.commercelayer.app/<b>&lt;order-id&gt;</b>?accessToken=<b>&lt;your-access-token&gt;</b>
  </span>
</p>

This component can be used as in-place replacement for an `<a>` tag pointing to that url.

<Canvas withSource="open">
  <Story story={Basic} name="Basic" />
</Canvas>

ℹ️ Checkout link is **disabled** when cart does not contain shoppable items.

As soon as you <cl-add-to-cart class="mx-1 px-2 py-1" code={codes.available}>add to cart</cl-add-to-cart> a shoppable item, then the checkout link will be cliccable.
