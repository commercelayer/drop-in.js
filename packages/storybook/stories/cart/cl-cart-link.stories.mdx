# `cl-cart-link`

Any Commerce Layer account comes with a [hosted version of the Cart](https://github.com/commercelayer/commercelayer-cart#hosted-version) application that is automatically enabled.

<p>
  You can use the hosted version of the Cart application with the following URL format:
  <span class="px-2 py-1 border border-[#EEEEEE] bg-[#F8F8F8] rounded text-[#464646] !text-[13px] !font-mono whitespace-nowrap inline-block">
    https://<b>&lt;your-organization-slug&gt;</b>.commercelayer.app/cart/<b>&lt;order-id&gt;</b>?accessToken=<b>&lt;your-access-token&gt;</b>
  </span>
</p>

This component can be used as in-place replacement for an `<a>` tag pointing to that url.
