### `Minicart`

The `cl-cart` behaves like a minicart when rendered inside a [**cl-cart-link**](?path=/docs/components-cart-cl-cart-link--basic).

This minicart is **unstyled and hidden by default**.
You'll need to write some css in order to make it work or you can just import the [**minicart.css**](?path=/docs/getting-started--page#minicartcss) file in your website.

```html dark
<link href="https://cdn.jsdelivr.net/npm/@commercelayer/drop-in.js@1.0.0-beta.1/dist/drop-in/minicart.css" rel="stylesheet" />
```
